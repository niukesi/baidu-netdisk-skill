## 🚀 快速开始

### 1. 安装

```bash
npx skills install github:niukesi/baidu-netdisk-skill
```

### 2. 获取百度 API 密钥

访问 [百度网盘开放平台](https://pan.baidu.com/union/apply) 申请开发者账号（企业审核更快）。

### 3. 配置

```bash
npx baidu-netdisk-skill config \
  -k 你的 API_KEY \
  -s 你的 SECRET_KEY \
  -t 你的 ACCESS_TOKEN \
  -r 你的 REFRESH_TOKEN
```

> 💡 **不会获取 Token？** [查看详细教程](./docs/QUICKSTART.md)

### 4. 使用

```bash
# 查看用户信息
npx baidu-netdisk-skill whoami

# 列出根目录文件
npx baidu-netdisk-skill list /

# 列出指定目录（支持 4 层 + 深度）
npx baidu-netdisk-skill list "/我的教程/第一层/第二层"

# 搜索文件
npx baidu-netdisk-skill search "关键词"

# 获取下载链接（流式，不占空间）
npx baidu-netdisk-skill download <fs_id>
```
