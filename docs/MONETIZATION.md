## 支付流程

### 国内用户

1. 访问订阅页面（Stripe/Lemon Squeezy）
2. 选择套餐
3. 支付宝/微信支付
4. 获得激活码
5. 激活：
   ```bash
   npx baidu-netdisk-skill activate 你的激活码
   ```

### 国际用户

1. 访问订阅页面（Stripe）
2. 选择套餐
3. 信用卡支付
4. 获得 API Key
5. 配置：
   ```bash
   npx baidu-netdisk-skill config -k 你的 API_KEY
   ```

```javascript
// 简化的验证逻辑
async function verifyActivation(code) {
  const response = await fetch('https://your-server.com/verify', {
    method: 'POST',
    body: JSON.stringify({ code })
  });
  
  return response.ok;
}
```

### 订阅状态检查

```javascript
// 每次运行时检查
async function checkSubscription() {
  const apiKey = config.get('apiKey');
  if (!apiKey) return 'free';
  
  const status = await fetchSubscriptionStatus(apiKey);
  return status; // 'free' | 'personal' | 'pro' | 'enterprise'
}
```

## 成本控制

### 百度 API 成本

- 免费额度：每日 1000 次调用
- 超出后：¥0.001/次

**测算**：
- 100 个用户 × 每日 10 次 = 1000 次/日
- 刚好在免费额度内
- 超出部分成本很低

### 服务器成本

- 激活验证：Cloudflare Workers（免费）
- 支付处理：Stripe（2.9% + $0.30/笔）
- 客服：你自己（初期）

**总成本**：< 5% 收入

## 风险对冲

### 风险 1：百度 API 政策变化

**对策**：
- 多账号分散风险
- 快速扩展其他网盘（夸克、WPS）
- 建立用户沟通渠道，及时通知

### 风险 2：竞品出现

**对策**：
- 先发优势（第一个上架）
- 持续迭代（功能领先）
- 建立品牌（安全、省 Token）

### 风险 3：用户不愿付费

**对策**：
- 免费版限制合理（够用但不爽）
- 付费版价值明显（省心、省时间）
- 提供退款保障（30 天无理由）
